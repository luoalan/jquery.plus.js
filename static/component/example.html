1. one component defining file sould contain three parts; 2. the three parts are: 2.1. style dom (scoped css) innerHTML must begin with "{" and end with "}" you can use any style/less syntax the style will affect the main dom 2.2. normal dom (the main dom) the style and the script in this file will affect this dom 2.3. script dom (the entrance js) the script will run in a closure in the closure "this" point to the main dom(jquery object) this has a property named "zygote" which point to the original dom(jquery object) which was replaced by main dom this has a property named "container" which point to the parent dom(jquery object) which trigger the render function this has a property named "component" which point to the defining component this has a property named "cid" which point to the main dom attribute cid(a randomId for connect with scoped css) this has a property named "family_share" which will share with all instances born form same component
